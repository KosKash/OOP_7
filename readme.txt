Для работы приложения нужен python3
Чтобы установить Python пройдите по ссылке python.org
Для Windows - установить пакет, путем далее, далее, готово
Для MacOs также через универсальный установщик
Для Linux - установить, если не установлено(pkgconf, make, gcc) (для ubuntu, mint и т.д. через пакетный менеджер apt, для Arch
Manjaro через пакетный менеджер pacman) 
Далее скачать xz tarball, открыть директорию с архивом, в терминале ввести tar -xvf Python3.**, перейти в каталог cd Python3.**
    далее ввести ./configure --enable-optimizations
    далее ввести make -j* (* = количество ядер процессора)
    далее ввести sudo make altinstall и дождаться установки
    для удобства можно сделать alias для нужной версии python

Для использовния приложения перейти в директорию приложения, и в терминале ввести python3 main.py, и ввести выражение в терминал
